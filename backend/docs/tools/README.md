# LangChain 工具系统

CampusMind 使用 LangChain 的 StructuredTool 定义工具，支持 ReAct Agent 调用。

## 架构

```
┌──────────────────────────────────────────────────────────────┐
│                      ReactAgent                              │
│  ┌──────────────────────────────────────────────────────┐  │
│  │                    System Prompt                      │  │
│  │         + Tool Definitions (JWC, Library, etc.)       │  │
│  └──────────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌──────────────────────────────────────────────────────────────┐
│                    Tool Context                              │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐│
│  │  user_id    │  │  session_   │  │  is_authenticated   ││
│  │             │  │  manager    │  │                     ││
│  └─────────────┘  └─────────────┘  └─────────────────────┘│
└──────────────────────────────────────────────────────────────┘
                              │
          ┌───────────────────┼───────────────────┐
          ▼                   ▼                   ▼
    ┌──────────┐       ┌──────────┐       ┌──────────┐
    │   JWC    │       │ Library  │       │  Career  │
    │  Tools   │       │  Tools   │       │  Tools   │
    └──────────┘       └──────────┘       └──────────┘
```

## 工具类型

| 类型 | 是否需要登录 | 描述 |
|------|-------------|------|
| 教务系统 (JWC) | 是 | 成绩、课表、排名、等级考试 |
| 图书馆 (Library) | 否 | 图书搜索、位置查询 |
| 就业中心 (Career) | 否 | 宣讲会、招聘信息 |
| 办公网 (OA) | 是 | 校内通知 |
| RAG | 否 | 知识库检索 |
