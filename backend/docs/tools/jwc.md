# 教务系统工具 (JWC)

查询中南大学教务系统的成绩、课表、排名等信息。

## 工具列表

### jwc_grade - 成绩查询

查询学生的考试成绩。

**参数:**

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| user_id | string | 否 | 用户 ID，不填则使用上下文 |
| term | string | 否 | 学期，如 `2024-2025-1`，为空查询全部 |

**返回格式:**

```markdown
## 成绩查询结果

| 学期 | 课程名称 | 成绩 | 学分 | 课程属性 | 课程性质 |
|------|----------|------|------|----------|----------|
| 2024-2025-1 | 高等数学 | 95 | 4.0 | 必修 | 公基 |
```

---

### jwc_schedule - 课表查询

查询学生的课表。

**参数:**

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| user_id | string | 否 | 用户 ID，不填则使用上下文 |
| term | string | 是 | 学期，如 `2024-2025-1` |
| week | string | 否 | 周次，`0` 为全部周，默认 `0` |

**返回格式:**

```markdown
## 课表查询结果

| 课程名称 | 教师 | 周次 | 地点 | 星期 | 节次 |
|----------|------|------|------|------|------|
| 高等数学 | 张三 | 1-16 | 教学楼A101 | 星期一 | 1-2 |

> 学期第1周开始于: 2024-09-02日
```

---

### jwc_rank - 专业排名

查询学生的专业排名。

**参数:**

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| user_id | string | 否 | 用户 ID，不填则使用上下文 |

**返回格式:**

```markdown
## 专业排名结果

| 学期 | 总分 | 班级排名 | 平均分 |
|------|------|----------|--------|
| 2024-2025-1 | 85.5 | 5/50 | 82.3 |
```

---

### jwc_level_exam - 等级考试成绩

查询英语四六级、计算机等级考试等成绩。

**参数:**

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| user_id | string | 否 | 用户 ID，不填则使用上下文 |

**返回格式:**

```markdown
## 等级考试成绩

| 科目 | 笔试成绩 | 机试成绩 | 总分 | 笔试等级 | 机试等级 | 总等级 | 考试日期 |
|------|----------|----------|------|----------|----------|--------|----------|
| 大学英语四级 | 425 | - | 425 | 合格 | - | 合格 | 2024-06-15 |
```

---

## 认证要求

**需要登录**: 是

工具调用时会检查：
1. 用户是否已登录（`ctx.is_authenticated`）
2. 教务系统 Session 是否有效

Session 失效时返回: `教务系统会话已过期，请重新登录`

---

## 服务类

`JwcService` 封装了所有教务系统查询逻辑：

```python
class JwcService:
    def __init__(self, session_manager: UnifiedSessionManager):
        self.session_manager = session_manager

    async def get_grades(self, user_id: str, term: str = "") -> List[Grade]:
        ...

    async def get_schedule(self, user_id: str, term: str, week: str = "0"):
        ...

    async def get_rank(self, user_id: str) -> List[RankEntry]:
        ...

    async def get_level_exams(self, user_id: str) -> List[LevelExamEntry]:
        ...
```
