# 就业中心工具

查询中南大学就业信息网站的宣讲会、招聘信息。

## 工具列表

### career_teachin - 宣讲会查询

获取宣讲会信息列表。

**参数:**

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| zone | string | 否 | 校区筛选，留空显示全部 |

**校区选项:**

- 岳麓山校区
- 天心校区
- 杏林校区
- 潇湘校区

**返回格式:**

```markdown
## 宣讲会信息

### 公司名称
百度在线网络技术（北京）有限公司

📍 宣讲地点: 岳麓山校区 立德楼报告厅
📅 宣讲时间: 2024-10-15 14:00
🕌 需求专业: 计算机、软件工程
```

---

### career_campus_recruit - 校园招聘

获取校园招聘信息。

**参数:**

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| keyword | string | 否 | 关键词搜索，留空显示全部 |

**返回格式:**

```markdown
## 校园招聘信息

### 招聘公告标题
华为技术有限公司 2025 届校园招聘

🏙️ 工作城市: 深圳、北京、上海
📅 发布时间: 2024-09-01
📋 岗位: 软件工程师、硬件工程师、算法工程师
```

---

### career_campus_intern - 实习信息

获取实习信息。

**参数:**

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| keyword | string | 否 | 关键词搜索，留空显示全部 |

**返回格式:**

```markdown
## 实习信息

### 实习岗位标题
腾讯科技 2024 暑期实习招聘

🏙️ 工作城市: 深圳
📅 发布时间: 2024-06-01
📋 实习岗位: 前端开发、后端开发、产品经理
```

---

### career_jobfair - 招聘会查询

获取大型招聘会信息。

**参数:**

无

**返回格式:**

```markdown
## 招聘会信息

### 2024 年秋季大型招聘会

📍 举办地址: 湖南省人才市场
🏙️ 举办城市: 长沙
📅 举办时间: 2024-11-20 09:00
📋 招聘会类型: 大型综合招聘会
🏢 参会企业: 500+ 企业
```
