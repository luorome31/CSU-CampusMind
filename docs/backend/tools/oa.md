# 办公网通知工具 (OA)

查询中南大学办公网校内通知。

## 工具列表

### oa_notification_list - 通知列表

获取校内通知列表。

**参数:**

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| keyword | string | 否 | 关键词搜索 |
| page | int | 否 | 页码，默认 1 |
| page_size | int | 否 | 每页数量，默认 20 |

**返回格式:**

```markdown
## 校内通知

### 关于举办 2024 年秋季运动会的通知

📅 发布日期: 2024-09-15
📋 类别: 校园活动
👤 发布单位: 体育学院
🔗 链接: https://oa.csu.edu.cn/notice/xxx

---

### 关于 2024 年国庆节放假安排的通知

📅 发布日期: 2024-09-10
📋 类别: 学校通知
👤 发布单位: 校长办公室
```

---

### oa_notification_detail - 通知详情

获取通知详细内容。

**参数:**

| 参数 | 类型 | 必填 | 描述 |
|------|------|------|------|
| notification_id | string | 是 | 通知 ID |

**返回格式:**

```markdown
## 通知详情

### 关于举办 2024 年秋季运动会的通知

📅 发布日期: 2024-09-15
📋 类别: 校园活动
👤 发布单位: 体育学院

## 通知内容

各学院、各单位：

...
```

---

## 认证要求

**需要登录**: 是

OA 工具需要用户已登录办公网，调用时需要有效的 OA Session。

---

## 子系统标识

```python
Subsystem.OA = "oa"
OA_URL = "https://oa.csu.edu.cn/con/"
```
